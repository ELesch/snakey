# App Icons

This directory should contain PWA icons in the following sizes:

## Required Icons

- `icon-72x72.png`
- `icon-96x96.png`
- `icon-128x128.png`
- `icon-144x144.png`
- `icon-152x152.png`
- `icon-192x192.png`
- `icon-384x384.png`
- `icon-512x512.png`
- `maskable-icon-512x512.png` (with safe zone padding)
- `badge-72x72.png` (for notification badges)

## Shortcut Icons

- `shortcut-dashboard.png` (96x96)
- `shortcut-reptiles.png` (96x96)
- `shortcut-add.png` (96x96)

## Generating Icons

You can generate these icons from a single high-resolution source (1024x1024 recommended) using tools like:

- [PWA Asset Generator](https://github.com/nicothin/pwa-asset-generator)
- [Real Favicon Generator](https://realfavicongenerator.net/)
- [PWA Builder Image Generator](https://www.pwabuilder.com/imageGenerator)

## Icon Design Guidelines

The Snakey app icon should:
- Feature a friendly snake/reptile motif
- Use the primary green color (`#4a7c59`)
- Be recognizable at small sizes
- Work on both light and dark backgrounds

For the maskable icon, ensure important content is within the "safe zone" (inner 80% of the icon).
